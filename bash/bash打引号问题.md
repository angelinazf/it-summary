## 使用here doc 
* 转义问题终极解决方案 -> 完全不转义
### 在某个命令的参数处使用
```
echo "`cat <<'SQLEOF'
xxx''xxx'xxx'xx  123123    123123
abc'asdf"
$(dont-execute-this)
foo"bar"''
SQLEOF`"
```

### 赋值
```
VAR="`cat <<'VAREOF'
abc'asdf"
$(dont-execute-this)
foo"bar"''
VAREOF`"
```
