#### 方法1:
```
LOAD DATA LOCAL INFILE 'file_path' INTO TABLE `table_name`
```
速度最快的方法,问题是默认不开启
mysql服务器的默认设置不允许读本地文件,需要手动开启,但是据说有安全问题
