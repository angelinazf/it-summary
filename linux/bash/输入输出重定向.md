### 将stderr重定向到stdout
```
rm a 2>&1
```

### 将stderr和stdout都重定向到一个文件
```
rm a &>> a.txt
```

### 用vim查看输出结果(需要程序运行结束才看得到运行结果)
#### 查看输出 stdout
```
cat --help | vim -
```
#### 查看stdout + stderr
```
cat --help 2>&1 | vim -
```

### 边输出边保存到文件
```
cat --help | tee 1.txt
```

#### 将stdout和stderr都重定向到一个文件里面并且显示出来
```
cat --help 2>&1 | tee 1.txt
```
