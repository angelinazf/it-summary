## 批量修改文件名
### 替换递归替换当前目录下面的所有.txt文件,将其后缀修改为.md
```
for f in `find -name '*.txt'`; do 
  mv $f `dirname $f`/`basename $f .txt`.md ;
done;
```
### 目录下面的.md文件,在前面加上数字前缀,不递归
```
$i=0
for f in `find -name '*.md'`; do 
  $i+=1
  mv $f `dirname $f`/`basename $f .txt`.md ;
done;
```

## 解释
* find 递归找文件
* dirname 目录名
* basename 文件名,可以去掉某些后缀
